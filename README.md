# fzf-neoyank

An FZF source for neoyank.

## Usage

Open [FZF][fzf] showing the contents of the default register.

```vim
:FZFNeoyank
```

## Documentation

For complete docs, check [doc/fzf_neoyank.txt](doc/fzf_neoyank.txt), or

```vim
:help fzf-neoyank
```

[fzf]: https://github.com/junegunn/fzf
